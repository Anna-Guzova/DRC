# 🇺🇦 Ukrainian War Stats App

This is my first React Native project 👀

A simple Android app that shows daily statistics of russian army losses based on official Ukrainian sources.

## 📱 What this app does

The app displays war statistics in a clean and minimal UI.

You can:
* view total losses
* see daily updates (+ changes)
* switch between screens
* pick a specific date using a calendar

## 🧩 Screens

* **Home** — shows the latest stats and overall numbers
* **Calendar** — lets you select a date and see stats for that day

## ⚙️ Built with

* React Native
* API (data fetching)
* Basic mobile components

## 💡 Why I built this

This is a **learning project** where I explored:

* how React Native works
* how to fetch and display data from an API
* how to build simple mobile layouts
* how to structure a small app

Basically — my first step into mobile development 🚀

## 📌 Notes

* Android only (iOS not implemented)
* UI is simple on purpose — focused on functionality

##  Personal note

This project is important to me because it's my first real app built with React Native.
It may be simple, but it helped me understand the basics and start building something meaningful.
